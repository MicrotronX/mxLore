Open a terminal in any project and run this one command:

The skill installs the local proxy, downloads the mxLore skills from GitHub, and writes ~/.claude/mxMCPProxy.ini automatically. Takes about 30 seconds.

What if I don't have /mxSetup?

You haven't installed the mxLore skills yet. Run these two commands first:

curl -L -o /tmp/mx.zip https://github.com/MicrotronX/mxLore-skills/archive/main.zip && unzip -o /tmp/mx.zip -d ~/.claude/skills-tmp && cp -r ~/.claude/skills-tmp/mxLore-skills-main/mx* ~/.claude/skills/ && rm -rf ~/.claude/skills-tmp /tmp/mx.zip

Then restart Claude Code and run /mxSetup with your key.
